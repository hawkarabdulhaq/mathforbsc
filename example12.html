Direct Product — EN / HU
Bilingual definitions and worked example.
English EN Magyar HU
Definition: The direct product of sets A and B is the set of all ordered pairs where the first element comes from A and the second from B. Definíció: Két halmaz, A és B direkt szorzatán (Cartesian‑szorzatán) mindazon rendezett párok halmazát értjük, amelyek első tagja A‑ból, a második B‑ből való.
Notation: A × B Jelölés: A × B
Example: If A = {a, e, f}, B = {b, c} Példa: Ha A = {a, e, f}, B = {b, c}
Then A × B = {(a,b), (a,c), (e,b), (e,c), (f,b), (f,c)} Ekkor A × B = {(a,b), (a,c), (e,b), (e,c), (f,b), (f,c)}
Also: B × B = {(b,b), (b,c), (c,b), (c,c)} Továbbá: B × B = {(b,b), (b,c), (c,b), (c,c)}
Observation: If A has m elements and B has n elements, then |A × B| = m × n. Megjegyzés: Ha A‑nak m eleme van és B‑nek n eleme, akkor |A × B| = m × n.
How the counting works / Hogyan számoljuk
Short narrative (EN / HU)

English

  1. When you take the direct product A × B, you match every element of A with every element of B.
  2. Suppose |A| = m and |B| = n.
  3. Each element of A pairs with all n elements of B → one row produces n pairs.
  4. There are m rows (elements of A) → total pairs = m × n.

✅ Worked example: A = {a,e,f} (m = 3), B = {b,c} (n = 2) → |A × B| = 3 × 2 = 6.
Set of pairs: { (a,b), (a,c), (e,b), (e,c), (f,b), (f,c) }.

Magyar

  1. A direkt szorzat A × B azt jelenti, hogy A minden elemét párosítjuk B minden elemével.
  2. Tegyük fel, hogy |A| = m és |B| = n.
  3. A minden egyes eleme n elemmel párosítható B-ből → egy sor n párt ad.
  4. m sor van (az A elemei) → összes párok száma = m × n.

✅ Dolgozott példa: A = {a,e,f} (m = 3), B = {b,c} (n = 2) → |A × B| = 3 × 2 = 6.
Párok halmaza: { (a,b), (a,c), (e,b), (e,c), (f,b), (f,c) }.

Tip: Emphasize that order matters: in general (x,y) ≠ (y,x), and A × B and B × A list different pairs.
