ABDULHAQ Hawkar — Symmetric Difference of Sets

Slide 9 — Symmetric difference / Szimmetrikus differencia Geology context: exclusive rock types in two sets
English EN Magyar HU
Definition: The symmetric difference of sets A and B is the set of elements that are in A or in B, but not in both. Definíció: Két halmaz (A és B) szimmetrikus differenciáján azokat az elemeket értjük, amelyek A-ban vagy B-ben vannak, de nem mindkettőben.
Example: If A = {granite, basalt}, B = {basalt, limestone}, then A Δ B = {granite, limestone}. Példa: Ha A = {gránit, bazalt}, B = {bazalt, mészkő}, akkor A Δ B = {gránit, mészkő}.
Notation: A Δ B. Jelölés: A Δ B.
Example: {igneous rocks} Δ {sedimentary rocks} = all rocks that are igneous or sedimentary but not both. Példa: {magmás kőzetek} Δ {üledékes kőzetek} = minden kőzet, amely magmás vagy üledékes, de nem mindkettő.
Formula: A Δ B = (A \ B) ∪ (B \ A). Képlet: A Δ B = (A \ B) ∪ (B \ A).
Example: (A \ B) = {granite}, (B \ A) = {limestone}, so A Δ B = {granite, limestone}. Példa: (A \ B) = {gránit}, (B \ A) = {mészkő}, tehát A Δ B = {gránit, mészkő}.
The members of A Δ B are exactly those that belong to A or B, but not to both. Az A Δ B tagjai pontosan azok, amelyek A-ban vagy B-ben vannak, de nem mindkettőben.
Example: Fossil set A = {trilobite, ammonite}, fossil set B = {ammonite, coral}. Then A Δ B = {trilobite, coral}. Példa: Kövülethalmaz A = {trilobita, ammonitesz}, kövülethalmaz B = {ammonitesz, korall}. Ekkor A Δ B = {trilobita, korall}.

Teaching tip: Hide examples first; let students identify exclusive elements, then reveal symmetric difference and diagram.
