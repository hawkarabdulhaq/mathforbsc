Logic Assignment — Build the Formula

Feladat — Logikai képletalkotás

Instructions / Útmutató

EN: Write each statement below as a propositional-logic formula using only these symbols:
p, q, r, o, y, i (atoms) ¬ not ∧ and ∨ or → implies ↔ iff ( ) parentheses

Use ASCII if you prefer: ¬p = ~p, p∧q = p&q, p∨q = p|q, p→q = p->q, p↔q = p<->q.

Variables / Változók

  • p: EN the rock is sandstone • HU a kőzet homokkő
  • q: EN it contains fossils • HU fosszíliákat tartalmaz
  • r: EN it formed in a shallow marine environment • HU sekély tengeri környezetben képződött
  • o: EN the sample contains olivine • HU a minta olivint tartalmaz
  • y: EN the sample contains pyroxene • HU a minta piroxént tartalmaz
  • i: EN the sample is igneous • HU a minta magmás
Student tasks / Hallgatói feladatok
  1. EN: If the rock is sandstone and it contains fossils, then it formed in a shallow marine environment.
    HU: Ha a kőzet homokkő és fosszíliákat tartalmaz, akkor sekély tengeri környezetben képződött.
  2. EN: The rock is sandstone exactly when it contains fossils.
    HU: A kőzet akkor és csak akkor homokkő, ha fosszíliákat tartalmaz.
  3. EN: If it did not form in a shallow marine environment, then it is not sandstone or it does not contain fossils.
    HU: Ha nem sekély tengeri környezetben képződött, akkor nem homokkő, vagy nem tartalmaz fosszíliákat.
  4. EN: If the sample contains olivine or pyroxene, then it is igneous.
    HU: Ha a minta olivint vagy piroxént tartalmaz, akkor magmás.
  5. EN: Exactly one of these holds: the rock is sandstone, it contains fossils.
    HU: Pontosan az egyik igaz: a kőzet homokkő, a kőzet fosszíliás.
  6. EN: The sample contains neither olivine nor pyroxene.
    HU: A minta sem olivint, sem piroxént nem tartalmaz.
  7. EN: If the sample is igneous, then it contains olivine or pyroxene; and if it contains olivine, then it is igneous. (write both, joined by “and”)
    HU: Ha a minta magmás, akkor olivint vagy piroxént tartalmaz; és ha olivint tartalmaz, akkor magmás. (mindkettőt írd fel, „és”-sel összekötve)
  8. EN: If sandstone implies fossils, and fossils imply shallow marine, then sandstone implies shallow marine. (write this as one implication using parentheses)
    HU: Ha a homokkő implikálja a fosszíliát, és a fosszíliák implikálják a sekély tengert, akkor a homokkő implikálja a sekély tengert. (egyetlen → állítással, zárójelekkel)
  9. EN: Show that “If it contains olivine or pyroxene, then it is igneous” is equivalent to its contrapositive “If it is not igneous, then it contains neither olivine nor pyroxene.” (write a single ↔ formula)
    HU: Mutasd meg, hogy „Ha olivint vagy piroxént tartalmaz, akkor magmás” ekvivalens a kontrapozitívval: „Ha nem magmás, akkor sem olivint, sem piroxént nem tartalmaz.” (egy ↔ formulában)
  10. EN: It formed in a shallow marine environment iff it is sandstone and contains fossils.
    HU: Sekély tengeri környezetben képződött akkor és csak akkor, ha homokkő és fosszíliás.
