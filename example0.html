Slide 2 — The Concept of a Set / A halmaz fogalma
Bilingual bullets (EN/HU)
English EN Magyar HU
A set = a collection of objects taken as one whole A halmaz = dolgok összessége, egy egységként kezelve
The objects inside are called elements A halmazban lévő dolgok: elemek
“Set” and “element” are basic concepts (not defined further) „Halmaz” és „elem” = alapfogalmak (nem vezetjük vissza másra)
The branch of math that studies sets = Set Theory A halmazokkal foglalkozó matematikai ág: Halmazelmélet
• Founded by Georg Cantor (1845–1918) • Alapítója: Georg Cantor (1845–1918)
In naive set theory, a set can contain any kind of objects A naiv halmazelmélet szerint a halmaz elemei bármi lehetnek
A set is completely determined by its elements Egy halmazt az elemei teljesen meghatároznak
Sets do not change over time Halmazok nem változnak időben
