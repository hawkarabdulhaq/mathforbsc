ABDULHAQ Hawkar — Proper Subset (A ⊂ B) / Valódi részhalmaz (A ⊂ B)

Slide — Proper subset basics / A valódi részhalmaz alapjai
Bilingual bullets with geology‑flavored examples (toggle below)
English EN Magyar HU
A ⊂ B means: All elements of A are inside B, but A is not equal to B. A ⊂ B jelentése: A minden eleme B-ben van, de A nem azonos B-vel.
Example (geology): A = {granite}, B = {granite, basalt, obsidian} → then A ⊂ B because granite is in B, and B has more rocks. Példa (geológia): A = {gránit}, B = {gránit, bazalt, obszidián} → ekkor A ⊂ B, mert a gránit benne van B-ben, és B-ben több kőzet van.
If A ⊂ B, then A ⊆ B. Proper subset always implies subset. Ha A ⊂ B, akkor A ⊆ B. A valódi rész mindig rész is.
If A is a proper part of B, it’s also a part of B. Example: Granite ⊂ Igneous rocks → also Granite ⊆ Igneous rocks. Ha A B valódi része, akkor A B része is. Példa: Gránit ⊂ Magmás kőzetek → egyben Gránit ⊆ Magmás kőzetek.
A ⊂ A is never true (irreflexivity): a set cannot be a proper subset of itself; “proper” means “strictly smaller.” A ⊂ A sohasem igaz (irreflexivitás): egy halmaz nem lehet saját maga valódi részhalmaza; a „valódi” „szigorúan kisebbet” jelent.
Example: The Granite set is not a proper subset of the Granite set (they are equal). Példa: A Gránit‑halmaz nem valódi része a Gránit‑halmaznak (azonosak).
If A ⊂ B, then B ⊃ A (asymmetry): B is a proper superset of A. Ha A ⊂ B, akkor B ⊃ A (aszimmetria): B A valódi felülhalmaza.
Example: Granite ⊂ Igneous rocks → Igneous rocks ⊃ Granite. Példa: Gránit ⊂ Magmás kőzetek → Magmás kőzetek ⊃ Gránit.
If A ⊂ B and B ⊂ C, then A ⊂ C (transitivity): the proper‑subset relation passes through. Ha A ⊂ B és B ⊂ C, akkor A ⊂ C (tranzitivitás): a valódi részhalmaz reláció „átöröklődik”.
Example: A = {granite}, B = {granite, basalt}, C = {all igneous rocks} → A ⊂ B and B ⊂ C, so A ⊂ C. Példa: A = {gránit}, B = {gránit, bazalt}, C = {összes magmás kőzet} → A ⊂ B és B ⊂ C, tehát A ⊂ C.
👉 In short: Proper subset = “inside, but smaller.” It’s part of, but not equal. The four rules (implies subset, not itself, superset, transitivity) keep the idea consistent. 👉 Röviden: Valódi részhalmaz = „benne van, de kisebb.” Része, de nem azonos. A négy szabály (részhalmaz‑következmény, nem önmaga, felülhalmaz, tranzitivitás) teszi következetessé.
