🧠 Step 1: The setup / Kiindulás
Define simple statements and use symbols to make reasoning precise.
English ENMagyar HU
Let h = “It is hot.”    Let s = “It is sunny.”
We will translate full English sentences into symbolic logic. 		Legyen h = „Meleg van.”    Legyen s = „Napos az idő.”
Ezeket a mondatokat lefordítjuk logikai jelekre.
Symbols used here: ¬ (NOT), ` (AND), ~ (OR). You may also meet ∧ and ∨. Használt jelek: ¬ (NEM), ` (ÉS), ~ (VAGY). Előfordulhat még: ∧, ∨.
🔤 Step 2: Example (a) — “but” means “and” / A „de” logikailag „és”
“But” adds contrast in tone; the logic is still AND.
ENHU
“It is not hot but it is sunny.” → same as “not hot and sunny.”
Symbolic form: ¬h ` s (or ,h ` s). 		„Nem meleg van, de napos.” → ugyanaz, mint „nem meleg és napos.”
Jelölés: ¬h ` s (vagy ,h ` s).
🔤 Step 3: Example (b) — “neither … nor …” / „sem … sem …”
Means NOT the first AND NOT the second.
ENHU
“It is neither hot nor sunny.” → “not hot and not sunny.”
Symbolic form: ¬h ` ¬s (or ,h ` ,s). 		„Sem meleg nincs, sem napsütés.” → „nem meleg és nem napos.”
Jelölés: ¬h ` ¬s (vagy ,h ` ,s).

📏 Step 5: Inequalities as logic / Egyenlőtlenségek mint logika
Math “and/or” maps directly to logic.
ENHU
x ≤ a reads “x < a or x = a” → uses ~. x ≤ a jelentése „x < a vagy x = a” → ~.
a ≤ x ≤ b means “a ≤ x and x ≤ b” → uses `. a ≤ x ≤ b azt jelenti: „a ≤ x és x ≤ b” → `.
2 ≤ x ≤ 1 is false — no number satisfies both conditions. 2 ≤ x ≤ 1 hamis — nincs szám, amely mindkettőnek megfelel.
Geoscience example: “Grain size ≥ 0.5 mm and ≤ 2 mm” → sand‑sized → use `. Geós példa: „Szemcseméret ≥ 0,5 mm és ≤ 2 mm” → homokfrakció → `.
