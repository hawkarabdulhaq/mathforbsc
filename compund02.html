Translating from English to Symbols: “But” & “Neither–Nor” / Fordítás jelekre: „de” és „sem–sem”
Let h = “It is hot”, s = “It is sunny”. / Legyen h = „Meleg van”, s = „Napos az idő”.
English prompt EN Magyar feladat HU
Write each sentence symbolically (using ¬ or comma for NOT, ` for AND, ~ for OR). Írd fel a mondatokat jelöléssel (használható ¬ vagy vessző a NEM‑hez, ` az ÉS‑hez, ~ a VAGY‑hoz).
a) It is not hot but it is sunny. a) Nem meleg, de napos.
b) It is neither hot nor sunny. b) Sem meleg, sem napos.
Solution / Megoldás
Both ¬ and leading comma “,” are accepted for NOT.
English EN Magyar HU
a) “It is not hot but it is sunny” ≡ “not hot and sunny”.
Symbolic: ¬h ` s (or ,h ` s). 		a) „Nem meleg, de napos” ≡ „nem meleg és napos”.
Jelölés: ¬h ` s (vagy ,h ` s).
b) “Neither hot nor sunny” means “not hot and not sunny”.
Symbolic: ¬h ` ¬s (or ,h ` ,s). 		b) „Sem meleg, sem napos” azt jelenti: „nem meleg és nem napos”.
Jelölés: ¬h ` ¬s (vagy ,h ` ,s).
Inequalities as AND/OR statements / Egyenlőtlenségek ÉS/VAGY állításként
Assume x, a, b are particular real numbers. / Tegyük fel, hogy x, a, b konkrét valós számok.
English ENMagyar HU
x ≤ a means x < a or x = a (uses OR ~). x ≤ a jelentése: x < a vagy x = a (VAGY ~).
a ≤ x ≤ b means a ≤ x and x ≤ b (uses AND `). a ≤ x ≤ b azt jelenti: a ≤ x és x ≤ b (ÉS `).
Note: 2 ≤ x ≤ 1 is not satisfied by any real number
because it means 2 ≤ x and x ≤ 1, which can’t both be true. 		Megjegyzés: 2 ≤ x ≤ 1 egyetlen valós számmal sem teljesül,
mert azt jelenti: 2 ≤ x és x ≤ 1, ami nem lehet egyszerre igaz.
Why specify “real numbers”? / Miért „valós számok”?
English ENMagyar HU
Specifying x, a, b are real numbers ensures sentences like “x < a” and “x ≥ b” are either true or false — hence they are statements. Ha kikötjük, hogy x, a, b valós számok, akkor a „x < a” és „x ≥ b” típusú mondatok egyértelműen igazak vagy hamisak — vagyis állítások.
