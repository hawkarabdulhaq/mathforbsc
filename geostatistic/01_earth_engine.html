1) What you need first

  1. A Google account (Gmail or institutional).
  2. Decide your usage:
    • Noncommercial (free) for research, learning, NGOs, etc.
    • Commercial (paid) if you’ll use it for business. Google Earth Engine

1) Amire először szükség van

  1. Google‑fiók (Gmail vagy intézményi).
  2. Döntsd el a felhasználás módját:
    • Nem kereskedelmi (ingyenes) kutatáshoz, tanuláshoz, civil szervezeteknek stb.
    • Kereskedelmi (fizetős) ha üzleti célra használod. Google Earth Engine

2) Create your Earth Engine access

Option A — Quickest (Code Editor path)

  1. Go to the registration page and sign in.
  2. Click “Register a Noncommercial or Commercial Cloud project.”
  3. Choose Unpaid usage (noncommercial) or Paid usage (commercial), then follow the prompts to create or select a Google Cloud project and finish registration.
Note (2025): Noncommercial projects must complete a verification questionnaire (and periodic reverifications). Projects registered before Apr 16, 2025 need verification by Sept 26, 2025 to avoid a pause. Google for Developers

Option B — Through Google Cloud (good for org/commercial)

  1. In the Google Cloud Console, open the Earth Engine Product Registration page.
  2. Pick Noncommercial (unpaid) or Commercial (paid) and complete the form (questionnaire for noncommercial; choose a plan for commercial). Docs

2) Earth Engine hozzáférés létrehozása

A opció — A leggyorsabb (Code Editor útvonal)

  1. Nyisd meg a regisztrációs oldalt és jelentkezz be.
  2. Kattints: „Register a Noncommercial or Commercial Cloud project.”
  3. Válaszd a Unpaid usage (nem kereskedelmi) vagy a Paid usage (kereskedelmi) lehetőséget, majd kövesd a lépéseket Google Cloud projekt létrehozásához/kiválasztásához és a regisztráció befejezéséhez.
Megjegyzés (2025): A nem kereskedelmi projekteknek ellenőrzési kérdőívet kell kitölteniük (és időszakos újraellenőrzéseket). A 2025. ápr. 16. előtt regisztrált projekteknek 2025. szept. 26‑ig kell igazolniuk, különben a hozzáférés szünetelhet. Fejlesztői dokumentáció

B opció — Google Cloudon keresztül (szervezetek/üzleti cél esetén)

  1. A Google Cloud Console‑ban nyisd meg az Earth Engine Product Registration (termékregisztráció) oldalt.
  2. Válaszd a Noncommercial (ingyenes) vagy Commercial (fizetős) lehetőséget és töltsd ki az űrlapot (nem kereskedelmi esetén kérdőív; kereskedelmi esetén csomagválasztás). Dokumentáció

3) Open the Earth Engine Code Editor

Visit the Code Editor to confirm access and run scripts in your browser. It’s a web IDE with instant access to the data catalog.

3) Nyisd meg a Code Editor‑t

Látogasd meg a Code Editor felületet a hozzáférés ellenőrzésére és a böngészős futtatáshoz. Webes IDE az adat‑katalógus azonnali elérésével.

4) Your very first run (in the Code Editor)

  1. In the left Scripts pane, click New → Script.
  2. Paste this JavaScript and click Run:
// Load a Landsat 8 image over Budapest and display true color
var img = ee.ImageCollection('LANDSAT/LC08/C02/T1_L2')
  .filterBounds(ee.Geometry.Point(19.0402, 47.4979))
  .filterDate('2023-06-01', '2023-08-31')
  .sort('CLOUD_COVER')
  .first()
  .resample('bicubic');

var rgb = img.select(['SR_B4','SR_B3','SR_B2']).multiply(0.0000275).add(-0.2);
Map.centerObject(rgb, 10);
Map.addLayer(rgb, {min:0.03, max:0.3}, 'L8 True Color');

Any beginner quickstart/tutorial in the docs will also work the same way. Get Started

4) Az első futtatás (a Code Editor‑ban)

  1. A bal oldali Scripts panelen: New → Script.
  2. Illeszd be ezt a JavaScript kódot és kattints a Run gombra:

Bármely kezdő „Quickstart”/„Get Started” útmutató ugyanígy működik. Kezdő lépések

5) (Optional) Set up Python

Install the API:

pip install earthengine-api

Authenticate & initialize:

import ee
ee.Authenticate()     # follow the browser link, pick your Google Cloud project
ee.Initialize()       # choose/confirm the same project

# Quick test: print image info
img = ee.Image('CGIAR/SRTM90_V4')
print(img.getInfo()['id'])

Notebook auth now asks you to select a Google Cloud project during the flow. Auth docs

5) (Opcionális) Python beállítása

Common gotchas (and fixes)

  • “I registered but still can’t access” → Make sure your registration finished and is tied to the correct Google Cloud project, and complete the noncommercial questionnaire if applicable. Then re‑open the Code Editor. Access guide
  • Government/agency users → may need an institutional Google account and extra review for noncommercial access. Noncommercial

Gyakori buktatók (és megoldásuk)

  • „Regisztráltam, de mégsem férek hozzá” → Ellenőrizd, hogy a regisztráció befejeződött‑e, a megfelelő Google Cloud projekthez kötött‑e, és (ha kell) kitöltötted‑e a nem kereskedelmi kérdőívet. Ezután nyisd meg újra a Code Editort. Hozzáférés
  • Hatósági/szervezeti felhasználók → gyakran intézményi Google‑fiók és extra felülvizsgálat szükséges a nem kereskedelmi hozzáféréshez. Nem kereskedelmi
