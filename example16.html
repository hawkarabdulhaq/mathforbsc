Countable sets — the idea / Megszámlálható halmazok — az ötlet
Bilingual content (figures below remain English-only).
English EN Magyar HU
Imagine a lab collection of rocks. Each sample gets a number: 1, 2, 3, … Képzeljünk el egy laboratóriumi kőzet‑gyűjteményt. Minden minta kap egy sorszámot: 1, 2, 3, …
Big idea: If you can list elements one by one, the set is countable — even if it never ends. Lényeg: Ha az elemek sorba rendezhetők és felsorolhatók, a halmaz megszámlálható — akkor is, ha végtelen.
The naturals ℕ = {1,2,3,…} are the simplest infinite set. Evens {2,4,6,…} and odds {1,3,5,…} are the same “size” because you can pair each n with 2n or 2n−1. A természetes számok ℕ = {1,2,3,…} a legegyszerűbb végtelen halmaz. A páros {2,4,6,…} és a páratlan {1,3,5,…} számok „ugyanakkora” végtelenek, mert minden n-hez hozzárendelhető 2n vagy 2n−1 (bijekció).
Geology analogy: A fossil database numbered 1,2,3,…. The even‑numbered fossils (2,4,6,…) still form an infinite but countable subset. Geológiai analógia: Egy kövületadatbázis sorszámozva 1,2,3,…. A páros sorszámú kövületek (2,4,6,…) végtelen, de megszámlálható részhalmazt alkotnak.
Pairing with ℕ / Párosítás ℕ‑nel
Explanation is bilingual; diagrams stay English‑only.
English EN Magyar HU
A 1‑to‑1 matching (bijection) between ℕ and the evens uses the rule f(n)=2n. Similarly, odds use g(n)=2n−1. Az ℕ és a páros számok közti 1‑1 megfeleltetés (bijekció) a f(n)=2n szabállyal adható meg. A páratlanokra hasonlóan g(n)=2n−1.
These bijections prove evens and odds are countable and have the same cardinality as ℕ. Ezek a bijekciók bizonyítják, hogy a páros és páratlan számok megszámlálhatók, és ugyanakkora számosságúak, mint ℕ.
“Countable Sets I — Sequences” / „Megszámlálható halmazok I — Sorozatok”
Bilingual text; chart labels remain English‑only.
English EN Magyar HU
Mathematicians formalize “making a list” as a sequence. A sequence is a function a : ℕ → X; for each n ∈ ℕ we have a value an. A „felsorolást” a matematikában sorozatnak nevezzük. Egy sorozat függvény a : ℕ → X; minden n ∈ ℕ-hez tartozik egy érték an.
Geology analogy: Porosity along a borehole: 0 m → a₀=20%, 1 m → a₁=18%, 2 m → a₂=22%, … Geológiai analógia: Porozitás egy fúrólyuk mentén: 0 m → a₀=20%, 1 m → a₁=18%, 2 m → a₂=22%, …
If the borehole continued forever, you’d have an infinite sequence. If all values were 20%, the value‑set would be {20} (finite). With endlessly varying readings (at fixed precision), the value‑set is infinite and countable. Ha a fúrólyuk elvben a végtelenbe tartana, végtelen sorozatot kapnánk. Ha minden érték 20%, az értékhalmaz {20} (véges). Ha a mérések végtelenszer változnak (rögzített pontossággal), az értékhalmaz végtelen és megszámlálható.
Petrography — Point Counting / Petrográfia — Pontszámlálás
Bilingual text; the photo caption below is English‑only.
English EN Magyar HU
Imagine a granite thin section under a microscope. Granite commonly contains quartz, feldspar, biotite, muscovite. Question: How much of each mineral? Képzeljünk el egy gránit vékonycsiszolatot mikroszkóp alatt. A gránit gyakran tartalmaz kvarcot, földpátot, biotitot, muszkovitot. Kérdés: Mennyi van az egyes ásványokból?
Method — Point Counting: In petrography we view a grid of points (e.g., 1000). For each point, we record which mineral lies beneath. Módszer — Pontszámlálás: Petrográfiában pont‑rácson vizsgáljuk a mintát (pl. 1000 pont). Minden pontnál feljegyezzük, melyik ásvány található alatta.
Example sequence of identifications: Quartz, Quartz, Feldspar, Biotite, … This is a sequence a : ℕ → {Quartz, Feldspar, Biotite, Muscovite}. Azonosítási sorozat például: Kvarc, Kvarc, Földpát, Biotit, … Ez egy sorozat a : ℕ → {Kvarc, Földpát, Biotit, Muszkovit}.
Why math matters: By counting many points we estimate mineral percentages. As the number of counted points grows, the estimated proportions approach the “true” proportions in the rock. Miért fontos a matematika: Sok pont számlálásával becsüljük az ásvány‑százalékokat. Ahogy a pontok száma nő, a becsült arányok közelítenek a kőzet „valódi” arányaihoz.
✅ Takeaway / Összegzés
English EN Magyar HU
Countable infinity means you can number or list elements one by one. Megszámlálható végtelen azt jelenti, hogy az elemek sorszámozhatók, egyenként felsorolhatók.
Sequences are formal lists: a function ℕ → X. A sorozat formális lista: egy függvény ℕ → X.
Geology lens: Numbered samples, fossils, layers, or point‑count measurements naturally form countable sets and sequences. Geológiai nézőpont: A sorszámozott minták, kövületek, rétegek vagy pontszámlálási adatok természetesen megszámlálható halmazokat és sorozatokat adnak.
