1) Rule about functions (reminder)
A function just means: Every input gets exactly one output. That’s the only rule!		 1) Emlékeztető a függvényekről
A függvény annyit jelent: Minden bemenethez pontosan egy kimenet tartozik. Ennyi a szabály!
2) Outputs don’t have to be different
Think of mountains → highest rock type.
X: mountains in Hungary (Mátra, Bükk, Börzsöny).
Y: main rock types (andesite, limestone, …).

Example function:
Mátra → Andesite
Bükk → Limestone
Börzsöny → Andesite

Notice: Mátra and Börzsöny both map to “Andesite”. That’s allowed.		 2) A kimenetek ismétlődhetnek
Gondoljunk a hegység → uralkodó kőzettípus párosításra!
X: magyarországi hegységek (Mátra, Bükk, Börzsöny).
Y: fő kőzettípusok (andezit, mészkő, …).

Példa függvény:
Mátra → Andezit
Bükk → Mészkő
Börzsöny → Andezit

Megjegyzés: Mátra és Börzsöny is „Andezit” → ez teljesen rendben van.
3) Domain and codomain don’t have to be different
Try minerals → minerals (main alteration product).

Function f:
Feldspar → Clay (clay mineral)
Biotite → Chlorite
Quartz → Quartz (sometimes stays itself)

Here both domain and codomain are in the set of minerals — also fine.		 3) A tartomány és az értékkészlet lehet azonos típus
Példa: ásvány → ásvány (fő átalakulási termék).

Függvény f:
Földpát → Agyagásvány
Biotit → Klorit
Kvarc → Kvarc (néha önmaga marad)

Itt a tartomány és az értékkészlet is az ásványok halmazából való — ez is rendben.
4) The Big Idea
A function is flexible.
It only demands: one clear answer for each input.
It does not demand unique answers or different worlds for input/output.

In short:
• Different mountains can share the same dominant rock — still a valid function.
• A function can map minerals to minerals — domain & codomain may overlap.		 4) A lényeg
A függvény rugalmas fogalom.
Csak azt követeli meg, hogy minden bemenethez egyértelmű kimenet tartozzon.
Nem követeli meg, hogy a kimenetek különbözőek legyenek, vagy más „világból” jöjjenek.

Röviden:
• Különböző hegységek kaphatnak azonos kőzettípust — a függvény ettől még helyes.
• Függvény képezhet ásványt ásványra — a tartomány és értékkészlet átfedhet.

Unpacking the slide — terms, mapping language, notation (EN / HU)
Key words: arguments, values, image; write it as y = f(x).
English EN Magyar HU
1) New words: Elements of the domain (X) are arguments. Elements of the codomain (Y) are values (outputs). 1) Új szavak: Az értelmezési tartomány (X) elemei az argumentumok. Az értékkészlet (Y) elemei a függvényértékek (kimenetek).
2) “Mapping” language: The function f maps X to Y. We say Y is the image of X under f. 2) „Képezés”: A f függvény X‑ből Y‑ba képez. Y X képe f alatt.
3) Notation: If x ∈ X, then the function assigns one value f(x) ∈ Y. We write y = f(x). 3) Jelölés: Ha x ∈ X, akkor a függvény egy f(x) ∈ Y értéket rendel hozzá. Írjuk: y = f(x).
Geology example
X = {Granite, Basalt, Limestone}
Y = {Quartz, Pyroxene, Calcite}
Rule: f assigns each rock its main mineral.
f(Granite) = Quartz, f(Basalt) = Pyroxene, f(Limestone) = Calcite.
“Granite” is an argument; “Quartz” is a value. We can write Quartz = f(Granite). 		Geológiai példa
X = {Gránit, Bazalt, Mészkő}
Y = {Kvarc, Piroxén, Kalcit}
Szabály: f minden kőzethez a fő ásványt rendeli.
f(Gránit) = Kvarc, f(Bazalt) = Piroxén, f(Mészkő) = Kalcit.
A „Gránit” argumentum; a „Kvarc” érték. Írhatjuk: Kvarc = f(Gránit).
Examples of functions in geology • Why it matters (EN / HU)
Many outputs are measurements — numbers let us analyze processes.
EN — Examples
• Metamorphic rock → mineral composition (output: list of minerals)
• A site → number of fossils (output: integer count)
• Depth → porosity of sandstone (output: %)
• (Pressure, Temperature) → mineral phase (output: phase)
HU — Példák
• Metamorf kőzet → ásványos összetétel (kimenet: ásványlista)
• Helyszín → kövületek száma (kimenet: egész szám)
• Mélység → homokkő porozitása (kimenet: %)
• (Nyomás, Hőmérséklet) → ásványfázis (kimenet: fázis)
EN — Why it matters
Many important functions in Earth science have numeric outputs (measurements). Numbers let us describe and analyze natural processes mathematically.
HU — Miért fontos
A földtudomány sok fontos függvénye mérési eredményt ad (számot). A számok segítségével írhatjuk le és elemezhetjük a természetes folyamatokat.

Student takeaway / Lényeg: When you see y = f(x), think “the function takes an input (x) and gives one output (y).” In geology: “the rock type (x) determines the measured or named property (y).”
