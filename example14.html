Arrow diagram (English labels)
Each element of X has exactly one arrow to Y (function). Different X’s may point to the same Y.

Notes: A function may be many‑to‑one (different rocks could share the same dominant mineral), but never one‑to‑many for a single input. The image (range) is the set of actual outputs inside the codomain.
