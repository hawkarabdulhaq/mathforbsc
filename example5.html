ABDULHAQ Hawkar — Subsets & Proper Subsets

Slide 6 — Subsets / Részhalmazok Geology context: rock groups & samples
English EN Magyar HU
Definition: A set A is a subset of a set B if every element of A is also an element of B. Definíció: Egy A halmaz részhalmaza egy B halmaznak, ha A minden eleme eleme B-nek is.
Example: The set of {basalt, granite} is a subset of {basalt, granite, limestone, sandstone}. Példa: A {bazalt, gránit} részhalmaza a {bazalt, gránit, mészkő, homokkő} halmaznak.
Notation: A ⊆ B. Jelölés: A ⊆ B.
Example: {igneous rocks} ⊆ {all rocks}. Példa: {magmás kőzetek} ⊆ {összes kőzet}.
Every set is a subset of itself: A ⊆ A (reflexivity). Minden halmaz része önmagának: A ⊆ A (reflexivitás).
Example: {sandstone, shale} ⊆ {sandstone, shale}. Példa: {homokkő, pala} ⊆ {homokkő, pala}.
The empty set is a subset of every set. Az üres halmaz minden halmaznak része.
Example: ∅ ⊆ {granite, basalt}. Példa: ∅ ⊆ {gránit, bazalt}.
Definition: A set A is a proper subset of a set B if A is a subset of B but not equal to B. Definíció: Egy A halmaz valódi részhalmaza egy B halmaznak, ha A része B-nek, de nem azonos vele.
Example: {granite, basalt} ⊂ {granite, basalt, limestone}. Példa: {gránit, bazalt} ⊂ {gránit, bazalt, mészkő}.
Examples: • The set of prime numbers ⊂ natural numbers. • The set of parallelograms ⊂ quadrilaterals. Példák: • A prímszámok halmaza ⊂ a természetes számok halmazának. • A paralelogrammák halmaza ⊂ a négyszögek halmazának.
Example (geology): The set of {igneous rocks} ⊂ {igneous, sedimentary, metamorphic}. Példa (geológia): A {magmás kőzetek} ⊂ {magmás, üledékes, metamorf}.
Properties of ⊂ (proper subset): A ⊂ reláció tulajdonságai:
– If A ⊂ B, then A ⊆ B (proper subset implies subset). – Ha A ⊂ B, akkor A ⊆ B (a valódi rész mindig rész is).
Example: If {basalt} ⊂ {basalt, granite}, then {basalt} ⊆ {basalt, granite}. Példa: Ha {bazalt} ⊂ {bazalt, gránit}, akkor {bazalt} ⊆ {bazalt, gránit}.
– A ⊂ A is never true (irreflexivity). – A ⊂ A sohasem igaz (irreflexivitás).
Example: {granite, basalt} ⊄ {granite, basalt}. Példa: {gránit, bazalt} ⊄ {gránit, bazalt}.
– If A ⊂ B, then B ⊃ A (asymmetry). – Ha A ⊂ B, akkor B ⊃ A (aszimmetria).
Example: {basalt} ⊂ {basalt, granite} ⇒ {basalt, granite} ⊃ {basalt}. Példa: {bazalt} ⊂ {bazalt, gránit} ⇒ {bazalt, gránit} ⊃ {bazalt}.
– If A ⊂ B and B ⊂ C, then A ⊂ C (transitivity). – Ha A ⊂ B és B ⊂ C, akkor A ⊂ C (tranzitivitás).
Example: {basalt} ⊂ {igneous rocks}, {igneous rocks} ⊂ {all rocks} ⇒ {basalt} ⊂ {all rocks}. Példa: {bazalt} ⊂ {magmás kőzetek}, {magmás kőzetek} ⊂ {összes kőzet} ⇒ {bazalt} ⊂ {összes kőzet}.

Teaching tip: Hide examples first, let students try to state geological subsets, then reveal to confirm and expand.
