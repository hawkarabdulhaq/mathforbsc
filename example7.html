ABDULHAQ Hawkar — Difference of Sets

Slide 8 — Difference of sets / Halmazok differenciája Geology context: what remains when excluding rocks
English EN Magyar HU
Definition: The difference of sets A and B (written A \ B) is the set of all elements that are in A but not in B. Definíció: Az A és B halmazok differenciáján (A \ B) az A halmaz azon elemeinek halmazát értjük, amelyek nincsenek B-ben.
Example: If A = {granite, basalt, limestone}, B = {limestone, sandstone}, then A \ B = {granite, basalt}. Példa: Ha A = {gránit, bazalt, mészkő}, B = {mészkő, homokkő}, akkor A \ B = {gránit, bazalt}.
Notation: A \ B (“A minus B”). Jelölés: A \ B („A mínusz B”).
Example: {igneous rocks} \ {basalt} = {all igneous except basalt}. Példa: {magmás kőzetek} \ {bazalt} = {minden magmás bazalt nélkül}.
The elements of A \ B belong to A but do not belong to B. Az A \ B elemei benne vannak A-ban, de nincsenek benne B-ben.
Example: Mineral set A = {quartz, feldspar}, B = {feldspar}. Then A \ B = {quartz}. Példa: Ásványhalmaz A = {kvarc, földpát}, B = {földpát}. Ekkor A \ B = {kvarc}.

Teaching tip: Hide examples first, ask students to predict what remains after subtracting one rock set from another, then reveal.
