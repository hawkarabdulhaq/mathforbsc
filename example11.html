Granite Comparison — Ordered 10-tuple

EN / HU

Department of Atmospheric and Geospatial Data Sciences • Foundations of Set Theory

Overview / Áttekintés

We compare different granite samples.

Each granite contains common minerals: quartz, K-feldspar, albite, biotite, muscovite.

To describe one granite completely, we track for each mineral:

  • its amount (e.g., % quartz, % K-feldspar, …)
  • its characteristic grain size (how big the crystals are)

Altogether this makes 10 variables (5 minerals × 2 properties) — we can treat this as an ordered 10-tuple.

Különböző gránitminta-típusokat hasonlítunk össze.

Minden gránit gyakori ásványokat tartalmaz: kvarc, K-földpát, albit, biotit, muszkovit.

Egy minta teljes leírásához minden ásványhoz megadjuk:

  • mennyiséget (pl. % kvarc, % K-földpát, …)
  • jellemző szemcseméretet (mekkora kristályok)

Összesen ez 10 változó (5 ásvány × 2 tulajdonság) — ezt rendezett 10-esként ábrázoljuk.

From geology to math / Geológiából matematikába

In set theory, an ordered tuple like (a, b, c, …) captures both which values we have and their positions (1st, 2nd, …).

Unlike a plain set, order matters: swapping positions gives a different tuple.

For one granite sample, define the ordered 10-tuple as:

( qtz_amt, kfs_amt, alb_amt, bio_amt, mus_amt, qtz_size, kfs_size, alb_size, bio_size, mus_size )

That’s 10 slots in total — 5 amounts + 5 sizes.

A halmazelméletben a rendezett n-es (pl. (a, b, c, …)) a konkrét értékeket és a pozíciójukat is rögzíti (1., 2., …).

Nem úgy, mint egy egyszerű halmaznál, itt számít a sorrend: felcserélve másik n-est kapunk.

Egy gránitminta rendezett 10-ese:

( kvarc_menny, K-földpát_menny, albit_menny, biotit_menny, muszkovit_menny, kvarc_méret, K-földpát_méret, albit_méret, biotit_méret, muszkovit_méret )

Összesen 10 pozíció — 5 mennyiség + 5 méret.

Why this matters / Miért fontos?
  • This shows how ordered tuples model real data.
  • A granite is more than “a set of minerals” — it’s a structured collection of values.
  • We can compare samples scientifically: check all 10 positions.
  • Megmutatja, hogyan írhatók le valós adatok rendezett n-esekkel.
  • A gránit nem csak „ásványok halmaza”, hanem strukturált tulajdonsághalmaz.
  • Tudományos összehasonlítás: mind a 10 pozíciót ellenőrizzük.
