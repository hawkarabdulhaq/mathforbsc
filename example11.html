ABDULHAQ Hawkar — Granite comparison as an ordered 10‑tuple / Gránitminták rendezett 10‑esként

Granite samples (hand specimen & thin section views) / Gránitminták (kéziminta & vékonycsiszolat)
Slide — From minerals to a 10‑tuple / Ásványoktól a 10‑es rendezett n‑esig
We compare granite samples using amounts (%) and grain sizes of five minerals / Gránitmintákat hasonlítunk össze öt ásvány százaléka és szemcsemérete alapján
QZ = Quartz / Kvarc KF = K‑feldspar / K‑földpát AB = Albite / Albit BT = Biotite / Biotit MS = Muscovite / Muszkovit
English EN Magyar HU
We compare different granite samples. Különböző gránitmintákat hasonlítunk össze.
Each granite contains minerals like: quartz, K‑feldspar, albite, biotite, muscovite. Minden gránit olyan ásványokat tartalmazhat, mint: kvarc, K‑földpát, albit, biotit, muszkovit.
To describe one granite completely, we need to know:
  • The amount of each mineral (e.g. % quartz, % K‑feldspar, …).
  • The characteristic grain size of each mineral (how big the crystals are).
 Egy gránit teljes leírásához tudnunk kell:
  • Az egyes ásványok mennyiségét (pl. % kvarc, % K‑földpát, …).
  • Az egyes ásványok jellemző szemcseméretét (mekkorák a kristályok).
Altogether, there are 10 variables (5 minerals × 2 properties each = 10). These can be seen as an ordered 10‑tuple (like a 10‑item list). Összesen 10 változó van (5 ásvány × 2 tulajdonság = 10). Ez felfogható rendezett 10‑esként (10 elemű lista).
Breaking it down. In set theory, a tuple (like (a,b,c,…)) records order and position. Unlike a plain set, the first, second, … positions matter. Lebontva. A halmazelméletben a tuple (pl. (a,b,c,…)) a sorrendet és a pozíciót rögzíti. Egy egyszerű halmazzal ellentétben számít, melyik az első, második, stb.
So a granite sample can be represented as:
(QZ amount, KF amount, AB amount, BT amount, MS amount, QZ size, KF size, AB size, BT size, MS size)
That’s 10 slots in total. 		Tehát egy gránitminta így írható le:
(QZ százalék, KF százalék, AB százalék, BT százalék, MS százalék, QZ méret, KF méret, AB méret, BT méret, MS méret)
Összesen 10 „hely”.
Why this matters EN Miért fontos HU
This example shows how set theory and ordered tuples can describe real‑world data like rock composition. Each granite is not just “a set of minerals” but a structured collection of properties. Two granites are different if any of the 10 values differ. Ez a példa megmutatja, hogy a halmazelmélet és a rendezett n‑esek alkalmasak a valós adatok, például a kőzetösszetétel leírására. Egy gránit nem csupán „ásványok halmaza”, hanem tulajdonságok strukturált együttese. Két gránit akkor különbözik, ha a 10 érték közül bármelyik eltér.

Tip: read the tuple left‑to‑right, top row first (positions 1–5: amounts), then bottom row (positions 6–10: sizes). / Tipp: balról jobbra olvassuk, először a felső sor (1–5: százalékok), majd az alsó sor (6–10: méretek).
