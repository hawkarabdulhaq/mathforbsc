ABDULHAQ Hawkar — Union of Sets

Slide 7 — Union of sets / Halmazok uniója Geology context: combining rock and mineral groups
English EN Magyar HU
Definition: The union of two sets A and B is the set of all objects that are elements of A, or of B, or of both. Definíció: Két halmaz (A és B) unióján értjük mindazon dolgok halmazát, amelyek A-nak vagy B-nek, vagy mindkettőnek elemei.
Example: If A = {basalt, granite}, B = {limestone, sandstone}, then A ∪ B = {basalt, granite, limestone, sandstone}. Példa: Ha A = {bazalt, gránit}, B = {mészkő, homokkő}, akkor A ∪ B = {bazalt, gránit, mészkő, homokkő}.
Notation: A ∪ B. Jelölés: A ∪ B.
Example: {igneous rocks} ∪ {sedimentary rocks} = {igneous, sedimentary}. Példa: {magmás kőzetek} ∪ {üledékes kőzetek} = {magmás, üledékes}.
The elements of A ∪ B are exactly those belonging to A, or to B, or to both. Az A ∪ B elemei pontosan azok, amelyek A-nak, vagy B-nek, vagy mindkettőnek elemei.
Example: Fossil set A = {trilobite}, fossil set B = {ammonite, trilobite}. Then A ∪ B = {trilobite, ammonite}. Példa: Kövülethalmaz A = {trilobita}, kövülethalmaz B = {ammonitesz, trilobita}. Ekkor A ∪ B = {trilobita, ammonitesz}.
If A and B are finite sets with no common elements, then: Ha A és B véges, közös elem nélküli halmazok, akkor:
→ Number of elements in A ∪ B = (A) + (B). → Az A ∪ B elemszáma = (A) + (B).
Example: A = {granite, basalt}, B = {sandstone, shale}. Then |A ∪ B| = 2 + 2 = 4. Példa: A = {gránit, bazalt}, B = {homokkő, pala}. Ekkor |A ∪ B| = 2 + 2 = 4.
Identities of Union: Az unió művelet azonosságai:
– A ∪ B = B ∪ A (commutativity). – A ∪ B = B ∪ A (kommutativitás).
Example: {granite} ∪ {basalt} = {basalt} ∪ {granite} = {granite, basalt}. Példa: {gránit} ∪ {bazalt} = {bazalt} ∪ {gránit} = {gránit, bazalt}.
– A ∪ A = A (idempotency). – A ∪ A = A (idempotencia).
Example: {limestone, dolomite} ∪ {limestone, dolomite} = {limestone, dolomite}. Példa: {mészkő, dolomit} ∪ {mészkő, dolomit} = {mészkő, dolomit}.
– A ∪ ∅ = A. – A ∪ ∅ = A.
Example: {sandstone, shale} ∪ ∅ = {sandstone, shale}. Példa: {homokkő, pala} ∪ ∅ = {homokkő, pala}.
– (A ∪ B) ∪ C = A ∪ (B ∪ C) (associativity). – (A ∪ B) ∪ C = A ∪ (B ∪ C) (asszociativitás).
Example: ({basalt} ∪ {granite}) ∪ {limestone} = {basalt, granite, limestone} = {basalt} ∪ ({granite} ∪ {limestone}). Példa: ({bazalt} ∪ {gránit}) ∪ {mészkő} = {bazalt, gránit, mészkő} = {bazalt} ∪ ({gránit} ∪ {mészkő}).

Teaching tip: Hide examples first; ask students to imagine combining sets of rocks, then reveal the examples.
